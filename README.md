# YhaZt.github.io

Modern professional portfolio site — built with plain HTML & CSS, hosted on GitHub Pages.

## Features

- 🎨 Dark-themed responsive design
- 🛠 Tech stack showcase (Vue, React, Node.js, Laravel, Firebase, Supabase, and more)
- 📁 Featured projects section with source links
- ⏱ WakaTime coding activity integration
- 📊 GitHub stats and streak cards
- 📬 Contact / social links

## Customisation

| What to change | Where |
|---|---|
| Bio & headline | `index.html` → Hero section |
| Projects | `index.html` → Projects section (add/remove `<article class="project-card">` blocks) |
| Social links | `index.html` → Contact section (update `href` values) |
| WakaTime charts | `index.html` → WakaTime section (update `src` URLs with your share links) |
| Colours & fonts | `style.css` |
